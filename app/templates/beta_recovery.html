Beta recovery notes

This page is read-only. It explains where SamWizard writes managed settings and where backups may exist.

Use these notes only if beta setup fails or you need to inspect what changed. SamWizard does not run rollback commands from this page.

Persistent drive mounts

SamWizard-managed mount blocks live in /etc/fstab.

sudo cp /etc/fstab.samba-wizard.YYYYMMDDHHMMSS.bak /etc/fstab
sudo systemctl daemon-reload

Samba shares

SamWizard-managed share blocks live in /etc/samba/smb.conf.

sudo cp /etc/samba/smb.conf.samba-wizard.YYYYMMDDHHMMSS.bak /etc/samba/smb.conf
sudo testparm -s
sudo systemctl restart smbd

Wi-Fi Netplan

The beta disables Wi-Fi setup. Older test runs may have used /etc/netplan/99-samba-wizard-wifi.yaml.

sudo ls -l /etc/netplan/99-samba-wizard-wifi.yaml*
sudo netplan generate
sudo netplan apply

Managed block markers

Look for these comments before making manual edits.

# BEGIN SAMBA WIZARD MOUNT
# END SAMBA WIZARD MOUNT
# BEGIN SAMBA WIZARD SHARE
# END SAMBA WIZARD SHARE
